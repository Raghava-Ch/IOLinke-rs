# IO-Link Device Stack

A modular, maintainable, memory safe and portable IO-Link Device/Slave stack implementation compliant with IO-Link Specification Version 1.1.4 (June 2024).

## Project Status
Heavy work in progress.
Development of this project is ongoing. The implementation is written in Rust and is designed for interoperability with C language environments.
